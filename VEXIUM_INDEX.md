## � Known Limitations & Roadmap

### Current Constraints (v1.0)

- ❌ **Single inheritance only** - No multiple inheritance yet
  - **Impact:** Limits composition patterns for AI/ML
  - **Solution:** Use composition with delegation
  - **See:** [MULTIPLE_INHERITANCE_PROPOSAL.md](MULTIPLE_INHERITANCE_PROPOSAL.md) for v2.0 plans

- ⚠️ **Limited module system** - Basic imports only
  - **Planned:** Better namespacing in v2.0

- ⚠️ **No async/await** - Synchronous execution only
  - **Planned:** Concurrency support in future

### Planned for v2.0 🎯

| Feature | Status | Impact | ETA |
|---------|--------|--------|-----|
| **Multiple Inheritance** | RFC ✅ | High for AI/ML | Q3 2026 |
| **Generics** | Proposed | Medium | Q4 2026 |
| **Interfaces** | Proposed | Medium | Q4 2026 |
| **Async/Await** | Planned | High | 2027 |
| **Package Manager** | Planned | Medium | Q1 2027 |

### How to Stay Updated

1. Read [MULTIPLE_INHERITANCE_PROPOSAL.md](MULTIPLE_INHERITANCE_PROPOSAL.md)
2. Check release notes in repository
3. Follow development in GitHub discussions
4. Contribute ideas and feedback
